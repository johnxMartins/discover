# Vantagens de usar o nodeJS

- Rápido

  - Execução
  - Prototipar

- Alta escaabilidade

- Aplicações de ponta

- JS everywhere

- Ecosistema gigante

- Casos de usos
  - Netflix (Usavam java do backEnd e javaScript no frontEnd)
  - Paypal

# O que é ?

- É um ambiente de execução
- Uma plataforma em que é possível criar aplicações JavaScript sem
